<h1 align="center">IT Onboarding System</h1>

<p align="center">
  A web-based application that streamlines the employee onboarding process by managing candidate registration, document verification, task tracking, and user role management.
</p>

<h2>📌 Project Overview</h2>
<p>
The IT Onboarding System is designed to automate and simplify the onboarding process for organizations. It helps HR teams and managers efficiently manage new employee information, verify documents, assign onboarding tasks, and monitor progress through a centralized platform.
</p>

<h2>🚀 Features</h2>
<ul>
  <li>Employee Registration and Profile Management</li>
  <li>Document Upload and Verification</li>
  <li>Role-Based Access Control (Admin, Manager, Employee)</li>
  <li>Onboarding Task Assignment and Tracking</li>
  <li>Employee Dashboard</li>
  <li>Manager Dashboard</li>
  <li>Secure Authentication and Authorization</li>
  <li>Responsive User Interface</li>
</ul>

<h2>⚙️ Installation</h2>
<ol>
  <li>Clone the repository</li>
  <li>Configure PostgreSQL database</li>
  <li>Update database credentials in <code>application.properties</code></li>
  <li>Run the Spring Boot application</li>
  <li>Open the application in your browser</li>
</ol>
